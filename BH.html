B&H

This project is focus on a redesign on B&H.

OVERVIEW

Nowadays, many people like to buy things online instead onsite, so e-commercial websites are the ones that were visited most frequently among all websites. Among those, there is a very busy website type, which has very high traffic since we had e-commercial websites - electronic e- commercial websites, like BestBuy, Newegg, Apple, B&H, BOSE, Microsoft, etc.
In this project, I will focus on a redesign on B&H.

PROBLEM

VISIBILITY OF SYSTEM STATUS

When create a new account and set up password, it gives a hint to tell the user what is missing.

USER CONTROL AND FREEDOM

The website provide good user control and freedom. When a user try to log into the website. It not only provides error prevention with a forgot option, but also provide the option that login with google account and facebook account that match the web world’s standards and consistency.



RECOGNITION

In production page. It has add to Cart, add to wish list and change quantity in similar position of other websites. This helps with minimize user’s memory load. While on this page, we can add small buttons around QTY to let a user increase and decrease quantity with mouse.

MATCH BETWEEN SYSTEM AND REAL WORLD

There is a drop off category on the top of homepage, which is really confusing and is not well match real world. This part should be moved to the bottom of the homepage and put more significant information on top.

SOLUTION

FLEXIBILITY AND EFFICIENCY OF USE

For some frequent users, a one-click-buy button on the production page can be really helpful. In this project, we will redesign the production page with adding such a button. Another design in this project is to add a comparison option on all pages. So that a user can add any product to be compared.



AESTHETIC AND MINIMALIST DESIGN

On the home page, there are two many categories on the navigation bar which is not only very confusing, but also make it difficult to let a user finds what a user needs.

Change the way of how we arrange three sections on production page. A puts the images on left side. B puts images in the middle of three sections.
